# CrestHill University Website

A modern and responsive university landing page built using HTML, CSS, Bootstrap, JavaScript, and AOS animations. The website highlights university programs, admissions, campus life, placements, and contact information with a clean and interactive UI.

---

## Features

- Responsive Navigation Bar
- Animated Hero Section
- Dynamic Programs Section
- Admissions Form
- Campus Life Carousel
- Placement Statistics Counter
- Company Logo Marquee
- Contact Section with Google Maps
- AOS Scroll Animations
- Responsive Footer
- Bootstrap Modals

---

## Technologies Used

- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- Bootstrap Icons
- AOS Animation Library
